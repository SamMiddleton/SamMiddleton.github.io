Intro to Javascript

What is JavaScript?

If a website was a robot arm, HTML would be the trusses that make up its structure, CSS would be panelling that makes it look nice and JavaScript would be the motors that make it move.

Javascript provides interactivity to websites, where HTML provides the content and structure and CSS provides the styling.

Control Flow and Loops

To do complex interactivity, it is often necessary to do things only sometimes, for example IF the laundry basket is full THEN put a wash on.

It is also useful to choose between options, e.g. IF more than half the dirty laundry is white THEN put a white wash on ELSE put a coloured wash on

You can also do things several times with loops, such as FOR EACH piece of clothing IN the clean laundry DO fold it

Another form of loop is the while loop, which could look like this: WHILE the bedsheet is more than 50cm on any side DO fold it in half

The Document Object Model

The Document Object Model, or DOM is a representation of the webpage's structure in a form that can be understood and manipulated by JavaScript code.

The following code, for example, finds the element with the ID of foo and sets the function bar to be called whenever the element is clicked (functions are explained below).

el = document.querySelector('#foo')
el.addEventListener('click', bar)

Arrays and Objects

There are two basic ways to store a collection of things in JavaScript: Arrays and Objects.

An array is a list of values, e.g. arr = ['foo', 'bar', 'baz'].

Elements of an array can be retrieved and modified by indexing, counting the elements from 0 so a = arr[1] assigns 'bar' to a and arr[2] = 'foobar' sets the last element of the array to 'foobar'.

Objects store information in key-value pairs like so: obj = {a:1, b:1, c:2, d:3, e:5} and are accessed like foo = obj.d.

Functions

A function is a self contained block of code that can be reused, e.g.

function collatz(num) {
  let result
  if (num % 2 === 0) {
    result = num / 2
  } else {
    result = 3*num + 1
  }
  return result
}

If we then call the function like a = collatz(10), then num is assigned the value 10 and the function's code is executed.

As 10 has a remainder of 0 when divided by 2, the first branch of the if-else is executed and result is assigned the value 5.

We then return result so the function exits and 5 gets assigned to a.

Functions are useful because they save us from having to write out the same code again and again, and they can abstract away implementation details from higher-order logic.
